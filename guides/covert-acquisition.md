## What is Covert Acquisition?
Yuri Grom uses **Covert** methods for both expansion (acquisition) and undermining. This means gathering intelligence data and delivering it — a non-combat activity that anyone can contribute to regardless of their combat skill.

## Preparation Phase
Before a system can be expanded into, it must be **prepared** during the preparation phase of the cycle. For Grom, preparation involves covert data collection missions from Power Contacts, then delivering them to the target system.
1. Check Discord for the current preparation targets (posted by coordinators at cycle start).
2. Visit a Yuri Grom Power Contact and accept **Covert Intelligence** pickup missions.
3. Deliver the data packages to a station in the target preparation system.
4. Each delivery contributes to the preparation score.

## Expansion Phase
If a system reaches the preparation threshold, it becomes an expansion candidate. CMDRs then run expansion missions from the Yuri Grom HQ area to the target system to complete the takeover.

> **Coordination is everything.** Covert acquisition only works when many CMDRs target the same system simultaneously. Always follow the coordinator-posted target list — solo preparation rarely reaches the threshold.

## What Ships to Use
Covert missions carry small cargo payloads, so ship size matters less than jump range. Use a fast, high-jump ship like a **Diamondback Explorer, Asp Explorer**, or engineered **Hauler** to minimise travel time between runs.

## Checking Targets
Expansion targets are posted weekly on the Yuri Grom Discord. Never prepare an unlisted system — it wastes your effort and may conflict with consolidation plans.
